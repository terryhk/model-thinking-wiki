# 第11章 设计学：以人为本的问题解决

---

📍 **破除焦虑**：「设计学是设计师的事，和我有什么关系？」→ 设计思维不是「设计好看的东西」，是「站在用户的角度发现问题，创造性地解决问题」——这个能力在任何职业中都有价值。

💡 **一句话心法**：先同理，再定义，再发散，再收敛，再验证——五步循环，以用户反馈为导向。

📊 **学科之眼**：设计思维教我们以人为本、用同理心解决问题。
