# 终章：格栅的组合力量

---

我们走过了121个思维模型。

从物理学的熵增与临界点，到数学的复利与概率；从生物学的自然选择与自催化，到心理学的认知误判与双系统；从经济学的机会成本与反脆弱，到商学的安全边际与护城河；从社会学的社会认同与结构洞，到哲学的第一性原理与逆向思维。八大学科，121个模型，每一个都是理解世界的一扇窗。

但121扇窗的价值，从来不在你能从每一扇窗看到什么——而在你同时打开几扇窗时，那些交叉的光线照亮了什么。

芒格说：「当你只有一把锤子，所有问题看起来都像钉子。」这句话的深意不只是「你需要更多工具」——而是说，单一工具不仅解决不了问题，还会扭曲你对问题本身的认知。只有锤子的人，看不到螺丝和铰链的区别；只有复利思维的人，看不到均值回归的力量；只有均值回归思维的人，看不到复利的奇迹。

格栅的力量，不在单个模型，在模型的组合。

这一章，是全书的收束，也是格栅理论的终极命题：如何让多模型协同工作，产生远超单个模型之和的力量？

## 尾声：格栅是终身修炼

121个模型，8个学科，4个章节——你读到了这里。

这不是结束，是开始。

思维模型不是一本读完就合上的书，而是一套需要终身修炼的操作系统。你今天读完了这本书，如果你不刻意练习、不持续迭代、不在真实决策中反复调用，下周你就会忘掉90%——这不是记忆力的问题，而是大脑的使用性依赖原则：不用就退化，常用才强化。

芒格95岁还在学习新的思维模型。段永平在每一次投资决策中仍然坚持检查清单——不是因为记不住，而是因为人永远不可能对自己的认知偏差免疫。清单不是对记忆的不信任，是对人性的诚实。

我想用两句话作为全书的结尾——

第一句来自芒格：

> 我这辈子一直在做的事，就是让自己有资格拥有更多的思维工具，而不是只有一把锤子。

第二句来自段永平：

> 做对的事情，然后把事情做对。前者是方向，后者是方法。方向比方法重要——但如果你没有方法，你连方向对不对都不知道。

思维模型给你的，正是判断「方向对不对」的方法。而方向，值得用一生去追寻。

格栅之门已经打开，路在你的脚下。
