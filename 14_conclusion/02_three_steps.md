# 三步构建法


理解了Lollapalooza效应，下一个问题是：我如何构建自己的格栅？

芒格给出的路径并不复杂，但需要耐心和纪律。我把它归纳为三步。

## 第一步：掌握各学科的大想法

**核心原则**：不是每个学科的所有知识，而是每个学科最核心的3-5个模型——学科中「大想法」级别的洞见。

芒格说：「你不需要成为一个物理学家，但你必须理解熵增意味着什么；你不需要成为一个心理学家，但你必须知道人的判断在哪里会系统性出错。」这就是「大想法」的标准——不是学科的全部，而是学科中对跨领域思维最有迁移价值的核心洞见。

以下是全书17个核心模型清单：

| 学科 | 核心模型 | 核心洞见 |
|------|---------|---------|
| 物理学 | 熵增定律（PC-1） | 封闭系统趋于混乱，必须主动输入负熵 |
| 物理学 | 临界点（PC-6） | 量变到质变的拐点，临界前的耐心与临界后的爆发 |
| 数学 | 复利（MT-1） | 指数增长的底层逻辑，时间是最大的杠杆 |
| 数学 | 概率与费马帕斯卡（MT-4） | 用期望值决策，概率×赔率才是完整的判断 |
| 生物学 | 自然选择（BI-1） | 适者生存不是最强者生存，而是最能适应环境者生存 |
| 生物学 | 自催化（PC-3） | 产出反哺投入，自增强循环一旦启动难以停止 |
| 心理学 | 认知误判（PS-1） | 人的判断有25种系统性偏差，自我纠偏是终身功课 |
| 心理学 | 卡尼曼双系统（PS-11） | 快思考与慢思考，直觉与推理的分工与协作 |
| 经济学 | 机会成本（EC-1） | 选择的代价不是你付出了什么，而是你放弃了什么 |
| 经济学 | 反脆弱（EC-6） | 从冲击中获益——不是承受打击，而是因打击而更强 |
| 商学 | 安全边际（BU-1） | 为不可避免的错误留出缓冲——对了多赚，错了少亏 |
| 商学 | 护城河（IN-3） | 长期竞争优势的五维度检验——品牌、网络、转换、规模、专利 |
| 社会学 | 社会认同（SO-1） | 人会不由自主地跟随群体——从众是最古老也最危险的偏差 |
| 社会学 | 结构洞（SO-2） | 占据网络节点位置的人拥有信息优势和控制优势 |
| 哲学 | 第一性原理（PH-2） | 回归最基本的命题，绕过未经检验的假设 |
| 哲学 | 逆向思维（PH-6） | 从对立面思考——如果我知道会死在哪里，就不去那里 |
| 组织理论 | 局部最优与全局最优（OT-1） | 优化的陷阱——眼前最好的选择可能让你永远到不了全局最优 |

17个模型，8个学科。这不是一个需要死记的清单，而是一张地图——你不需要记住地图上的每一条路，但你需要知道每条路通向哪里，以及路与路之间的连接。

**具体操作建议**：

1. **每个模型一张卡片**：正面写模型名称和核心洞见（一句话），背面写一个你亲身经历的案例。不是书本上的案例——是你自己的案例。只有变成了你的故事，模型才是你的工具。
2. **不要急于求全**：先掌握5-7个最常用的，再逐步扩展。就像学语言——先掌握高频词汇，再学低频词汇，效率最高。
3. **警惕「伪掌握」**：能用模型名称做标签 ≠ 理解了模型。检验标准是：你能不能在不看任何资料的情况下，用大白话向一个外行解释这个模型，并且让他觉得「有道理」？如果做不到，你只是认识这个词，不是掌握了这个模型。

**常见误区**：

- **误区一：贪多嚼不烂**。试图同时掌握17个模型，结果每个都停留在「名词解释」层面。不如先深度掌握5个，再扩展。
- **误区二：把模型当标签**。会用「这是幸存者偏差」给别人的观点贴标签，但自己决策时完全想不起调用。模型是工具，不是装饰。
- **误区三：忽视反面**。只学模型的适用场景，不学模型的失效边界。一个不知道自己何时会失效的模型，比没有模型更危险。

## 第二步：建立模型间的连结地图

掌握单个模型只是格栅的地基，模型之间的连结才是格栅本身。每学一个新模型，问自己三个问题：

**问题一：它与我已知的哪些模型相连？**

复利连接概率（复利的回报率本质上是概率分布的期望值），概率连接认知误判（人对概率的直觉判断有系统性偏差），认知误判连接社会认同（从众是最常见的误判之一），社会认同连接飞轮效应（用户增长的网络效应部分来自社会认同），飞轮效应连接自催化（自增强循环的物理学原型）……一张连结地图就在追问中逐渐成形。

**问题二：它在什么场景下可以与其他模型组合使用？**

做投资决策时，安全边际 + 能力圈 + 费马帕斯卡 + 反脆弱 = 价值投资的完整防御体系。做职业选择时，机会成本 + 能力圈 + 幂律分布 + 反脆弱 = 不选「平均好」，选「极端好」的决策框架。做学习规划时，刻意练习 + 艾宾浩斯 + 临界点 + 杠铃策略 = 高效学习的系统方法。场景是模型组合的粘合剂——脱离具体场景谈模型组合，只是纸上谈兵。

**问题三：它与哪些模型看似矛盾但实则互补？**

这是最关键的追问，因为它直接通往更深层的理解。

**案例：均值回归 vs. 复利——看似矛盾的两大力量**

均值回归说：「涨久必跌，极端表现终将回归平均水平。」复利说：「增长会自我加速，越涨越快。」这两个模型指向完全相反的方向——一个说好的不会持续，一个说好的会越来越好。

矛盾吗？不矛盾。因为它们回答的是不同层面的问题：

- 均值回归适用于**围绕价值波动的价格**。股价涨得离谱，终将回归内在价值——这是均值回归的地盘。
- 复利适用于**长期增长的价值本身**。一家优秀公司的内在价值随时间复利增长——这是复利的地盘。

理解了这层区别，你就获得了一个强大的投资直觉：**短期看均值回归（价格），长期看复利（价值）。** 短期价格波动要回归均值，但长期价值在复利增长——两者不仅不矛盾，而且共同构成了价格与价值关系的完整图景。

类似的「矛盾互补」还有很多：
- 熵增 vs. 耗散结构——混乱是必然的，但开放系统可以对抗混乱
- 二八定律 vs. 幂律分布——两者描述同一分布的不同视角，前者看比例，后者看形状
- 系统1 vs. 系统2——不是哪个更好，而是在什么场景下该用哪个
- 安全边际 vs. 幂律分布——前者要你保守，后者要你敢于重仓极端机会——但重仓的前提正是足够的安全边际

**每一对矛盾的背后，都藏着一个更高层面的统一。** 追问矛盾，才是深度思考的开始。

## 第三步：决策时调用检查清单

掌握了模型、建立了连结，最后一步是把它们变成决策时自动调用的工具。芒格强烈推荐清单思维——飞行员起飞前不凭记忆检查，用清单；外科医生手术前不凭记忆检查，用清单。思维模型的使用者同理：重大决策前，用清单。

以下是一份12个关键问题的决策检查清单：

**▎自我认知**

1. **我是否在能力圈内？** ——如果我不真正理解这件事，我应该先学习，而不是先决策。
2. **我是否被认知误判影响？** ——检查社会认同、自视过高、确认偏差、剥夺超级反应这四个最常见的误判。
3. **我的判断是否受近期情绪干扰？** ——如果我在兴奋、愤怒或焦虑中，推迟24小时再决策。

**▎逻辑验证**

4. **逆向思维：什么情况下这个决策会失败？** ——把失败场景列出来，如果你说不清会怎么失败，说明你没想够。
5. **第一性原理：抛开所有既有框架，这件事最基本的事实是什么？** ——从事实出发，而不是从结论倒推。
6. **二阶效应：然后呢？** ——这个决策的直接后果之后，会有什么连锁反应？

**▎价值评估**

7. **安全边际是否足够？** ——如果我的判断有30%的误差，我还愿意做这个决策吗？
8. **机会成本是什么？** ——同样的资源（时间、金钱、注意力）放在其他地方，会不会更好？
9. **期望值是否为正？** ——概率 × 赔率的乘积是否大于零？如果为负，不做。

**▎系统风险**

10. **这是局部最优还是全局最优？** ——眼前最好的选择是否让我失去到达更好位置的可能？
11. **是否考虑了反作用力？** ——这个决策实施后，系统会如何做出反应来对抗我？
12. **如果错了，我能承受吗？** ——最坏情况下，我会出局吗？如果会，这个决策的仓位就必须缩减。

**使用方法**：

- 不需要每次决策都过12个问题。日常小决策，只过1、4、7三个核心问题。重大决策（投资、职业、健康），过全部12个。
- 清单的价值不在于答案，在于**强制你慢下来**。系统1的冲动决策往往在3秒内完成，而清单强制你启动系统2，哪怕只多花30秒，也能避免大量低级错误。
- 清单是可以进化的。随着你的实践，你会发现某些问题对你特别有效，某些问题不太相关——增删改，使之成为你个性化的决策工具。

**常见误区**：

- **误区一：清单太长**。超过15个问题，你就不愿意用了。精简到你能坚持每次都用的长度。
- **误区二：机械执行**。过清单时只是走过场，内心已经决定了答案。清单的价值在于真正的暂停和反思，不在于打勾。
- **误区三：只对大事用**。芒格说：「小决策的累积效应不亚于大决策。」日常中等重要的决策，也应该过3个核心问题。
