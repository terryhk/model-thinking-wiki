# 第6章 经济金融学：资源在稀缺中流动

---

📍 **破除焦虑**：「经济学就是供求曲线和股市K线图吧？我又不炒股，学这些干嘛？」→ 你每天都在做经济学决策——选择加班还是陪家人、买房还是租房、跳槽还是留下、花时间学A还是学B。经济学不是炒股指南，而是理解资源如何在稀缺中流动的底层逻辑。「稀缺」和「流动」这两个词，几乎可以描述你所有的人生抉择。

💡 **一句话心法**：某种东西的成本，是为了得到它所放弃的东西——你每做一个选择，都在用另一个可能性买单。

📊 **学科之眼**：经济金融学的独特视角在于「稀缺」和「选择」——资源永远是有限的，欲望永远超过供给，所以每一步行动都有代价。价格、成本、风险、回报，这些不是金融从业者的专属术语，而是人类社会行为的底层语法。掌握了这门语法，你看世界的分辨率会突然提高一档：为什么打车越来越贵？为什么公司越大越难转型？为什么你在亏损时反而更不想卖？答案全在这一章。

## 模型速览表

| 编号 | 模型名称 | 核心洞见 | 核心程度 |
|------|---------|---------|---------|
| EC-1 | 机会成本 | 为了得到它所放弃的最高价值替代品 | ⭐核心 |
| EC-2 | 路径依赖 | 一旦进入某路径就会沿其发展，即使更优选择也难以切换 | |
| EC-3 | 二八定律 | 80%的结果来自20%的原因——找到关键少数 | |
| EC-4 | 网络效应 | 产品价值随用户增加而增大，赢家通吃 | |
| EC-5 | 前景理论 | 确定盈利则规避风险，确定损失则冒险 | |
| EC-6 | 反脆弱 | 从冲击中受益，而非仅仅抵抗冲击 | ⭐核心 |
| EC-7 | 均值回归 | 涨得太猛的总会回落，跌得太惨的总会反弹 | |
| EC-8 | 复式簿记 | 每一笔变动记两笔——有借必有贷，借贷必相等 | |
| EC-9 | 供给和需求曲线 | 价格由供需的交点决定 | |
| EC-10 | 边际效益 | 每增加一单位投入带来的额外收益递减 | |

| EC-12 | 黑天鹅 | 极端影响的低概率不可预测事件 | |
| EC-13 | 灰犀牛 | 大概率且影响巨大但被集体忽略的事件 | |
| EC-14 | 规模效应 | 规模增大带来单位成本下降 | |
| EC-15 | 信息不对称 | 交易双方信息不平等，信息落差就是利润或损失 | |
| EC-16 | 信息传递 | 信息的流动速度和质量决定决策质量 | |
| EC-17 | 增长要素模型 | 增长取决于劳动力×资本×技术的组合效率 | |
