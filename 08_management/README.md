# 第8章 管理学：如何把事情做成

---

📍 **破除焦虑**：「管理学就是开会、做PPT、填表格吧？」→ 管理学的核心不是这些——GTD教你清空大脑，PDCA教你迭代改进，清单思维教你避免遗漏，卢曼卡片盒教你构建知识格栅。真正的管理学，是一门关于「如何把事情做成」的学问。它不关心你有多聪明，而关心你的聪明是否被可靠地执行。

💡 **一句话心法**：聪明人犯的错，往往是记忆和注意力犯的错——清单是最强大的抗遗忘武器。

📊 **学科之眼**：管理学教我们如何把事情做成——从目标设定到执行反馈，从个人效率到组织协同。物理学告诉你世界怎么运转，心理学告诉你人怎么运转，而管理学告诉你：怎么让事情按你的意图运转。它的独特视角是「执行」——一个完美的计划如果无法可靠执行，还不如一份粗糙但可执行的清单。
