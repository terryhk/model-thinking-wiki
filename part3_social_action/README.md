# 第三部分：人在社会中如何行动

> 理解资源、竞争、组织与需求的底层逻辑

---

第三部分解决一个问题：**当你理解了世界的法则和自己的认知，你如何在社会中行动？**

不是「社会怎么运作」——那是社会学的教科书。而是：资源在稀缺中如何流动？企业如何创造和捕获价值？组织如何把事情做成？人与人如何连结？需求如何被发现和满足？设计如何以人为本地解决问题？

这一部分的逻辑线，从宏观到微观：

- **经济金融学**（第6章）：资源在稀缺中如何流动？→ 供需、边际、机会成本、反脆弱、均值回归
- **商学**（第7章）：企业如何创造、交付和捕获价值？→ 护城河、飞轮效应、颠覆式创新、MVP、商业模式
- **管理学**（第8章）：如何把事情做成？→ PDCA、清单思维、渐进式总结、灰度管理
- **社会学**（第9章）：人与人如何连结？→ 结构洞、信任与协作
- **营销学**（第10章）：需求如何被发现和满足？→ 上瘾模型、需求三角、POEM模型
- **设计学**（第11章）：如何以人为本地解决问题？→ 设计思维、同理心与快速原型

读完第三部分，你会建立一套社会行动的工具箱——面对任何商业或组织问题，你会先问「这里的激励结构是什么？护城河在哪里？飞轮转起来了吗？人们在哪个需求层次？我是在用物理思维控制，还是在用生物思维迭代？」
