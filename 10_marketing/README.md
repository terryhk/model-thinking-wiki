# 第10章 营销学：需求如何被发现和满足

---

📍 **破除焦虑**：「营销就是打广告和做活动吧？」→ 营销的核心不是推销，是理解需求——缺乏感+目标物+消费者能力=需求，三要素缺一不可。大多数人把营销等同于推广，但推广只是最后一个动作。真正的营销在产品诞生之前就开始了：找到谁有痛点，设计什么来消除痛点，确认对方付得起。做不到这三步，再多的广告预算也是打水漂。

💡 **一句话心法**：有痛点没能力是空想，有能力没痛点是伪需求——真实需求三者缺一不可。

📊 **学科之眼**：营销学教我们理解需求如何被发现和满足——从用户痛点到产品定位。它不教你「怎么卖」，它教你「先搞清楚有没有人想买」。

## 核心模型深入

### MK-1 定位理论 ⭐

**1. 一句话理解**

在用户脑子里占据一个位置——想到某个品类就想到你。不是「做更好的产品」，而是「做第一个进入心智的品牌」。

**2. 模型提出者**

艾·里斯（Al Ries）和杰克·特劳特（Jack Trout），1981年出版《定位：争夺用户心智的战争》。定位理论的诞生有一个关键背景：20世纪70年代的美国正经历信息大爆炸——电视普及、广告泛滥、产品同质化严重。里斯和特劳特观察到一个根本性变化：商业竞争的主战场已经从工厂转移到了顾客的心智。你的产品好不好不再是关键，关键的是顾客的心智里有没有你的位置。

本书第7章从投资视角分析了定位理论作为品牌护城河的价值。本章从营销视角重新展开：定位不是防守工具，而是进攻策略——如何在消费者心智中抢占地盘、如何设计营销行动来巩固那个位置。

**3. 模型介绍/概念**

定位理论的核心命题有三个，每一个都反直觉。

**竞争在心智中，而非在市场中。** 传统营销认为竞争发生在货架和渠道上——谁的产品更好、谁的价格更低、谁的渠道更广，谁就赢。定位理论说：不，竞争发生在顾客的脑子里。顾客不是在比较产品，而是在比较脑海中的概念。你真的分得清可口可乐和百事可乐的口味差别吗？大多数盲测证明，普通消费者根本喝不出来。但一到超市，你还是会毫不犹豫地选那个「真正的可乐」。这就是心智位置的力量——它独立于产品本质，却能决定购买行为。

**成为第一比做好更重要。** 里斯和特劳特发现了一个被反复验证的规律：在任何一个品类中，第一个进入消费者心智的品牌，往往能长期占据领导地位——可口可乐是第一个可乐，赫兹是第一个租车公司，哈佛是第一个美国大学。后来者即使产品更好，也很难撼动第一的位置。这不是因为消费者愚蠢，而是因为心智有「先入为主」的认知惰性：一旦某个品类和一个品牌绑定了，改变这个绑定需要付出巨大的认知成本。所以定位的核心策略不是「我比你更好」，而是「我是某一类的第一个」——如果你不能在现有品类中做第一，就创造一个新品类做第一。

**占据品类词，就是占据心智。** 定位的最高境界是品牌名=品类名。说到「搜索」就想到百度/谷歌，说到「凉茶」就想到王老吉，说到「怕上火」就想到王老吉——当你占据了一个品类词，竞争对手的每一句广告都在帮你强化这个关联。王老吉不是在做「更好的饮料」，它是在做「预防上火的饮料」——它创造了一个新品类，然后成为这个品类里的唯一。这就是定位的威力：你不用在红海里和别人拼杀，你可以重新定义战场。

定位理论的营销洞见与投资视角形成互补：投资关注的是定位带来的护城河宽度（第7章BU-16），营销关注的是定位的构建过程——如何发现心智空位、如何设计信息狙击、如何防止定位被稀释。前者回答「定位值多少钱」，后者回答「如何建一个定位」。

**4. 核心要素**

第一，心智有限——人的短期能记住7±2个品牌，实际购买只考虑2-3个。你不需要打败所有对手，你只需要进入顾客的「考虑集」。如果顾客在某个品类只能想起三个品牌，你要做的就是让自己成为其中之一。

第二，先入为主——第一个进入心智的品牌拥有认知优势。这个优势不是线性的，而是指数级的：第一名的市场份额通常是第二名的两倍，第二名的市场份额通常是第三名的两倍。这就是为什么「成为第一」远比「做得更好」重要。

第三，差异化绑定——将品牌与一个词、一个概念、一个场景深度绑定。宝马绑定「驾驶乐趣」，沃尔沃绑定「安全」，联邦快递绑定「隔夜送达」。每个成功的定位都是一句话就能说清的。如果你的定位需要三句话来解释，它大概率不够锋利。

第四，定位一旦建立，不要轻易改变——但可以延伸。七喜在1990年代改变定位做可乐替代品，从细分品类第一变成了大品类的边缘品牌，几乎死掉。反观可口可乐，它的核心定位「真正的可乐」从未动摇，但它围绕这个核心做了大量延伸——零度可乐、樱桃可乐、健怡可乐——每一段延伸都在强化「可乐=可口可乐」这个心智绑定。

**5. 实践/运用领域**

在品牌建设中，定位是所有营销动作的起点。先定位置，再做传播——而不是先做传播，再 hoping 顾客给你一个位置。你可以用一句话测试定位是否清晰：「说到___就想到我们。」空格里填不进一个简洁的词，说明你的定位还没有聚焦。

在产品命名中，名字是定位的第一个载体。好名字自带定位信息——「支付宝」告诉你这是用来付钱的，「滴滴」模拟了喇叭声暗示打车，「得到」暗示知识正在被你获取。坏名字需要额外解释，而心智从来不会为需要解释的东西留位置。

在进入新市场时，不要试图在成熟品类中从零开始做第一，而是寻找或创造心智空位——一个还没有品牌占据的细分品类。元气森林没有去做「更好的可乐」，它做的是「0糖0脂0卡的气泡水」——在碳酸饮料和健康饮品之间，它开出了一个新品类，然后成为这个品类的第一。

在防御定位时，最危险的策略是品牌延伸过度。当一个品牌试图代表所有东西，它最终什么也代表不了。李宁曾经是中国运动品牌第一，但在品牌延伸到时尚、休闲多个领域后，核心定位被稀释，被安踏精准锁定「专业运动」反超。

**6. 与其他模型关系**

→ **护城河（IN-3）**：品牌护城河是定位理论的投资学映射——定位在心智中的品牌拥有最宽的护城河。第7章BU-16从投资视角论述了定位的护城河价值，本章补充了营销视角的构建方法。

→ **波特一般战略（BU-13）**：定位理论是波特聚焦战略在营销领域的表达——不做全科冠军，做单项第一。波特说在成本领先、差异化、聚焦中选一个执行到底，定位理论说在顾客心智中选一个词占据到底，底层逻辑完全一致。

→ **需求三角（MK-3）**：定位回答的是「目标物是什么」——你在需求三角中选哪个缺乏感、提供什么解决方案，本身就是定位决策。定位和需求三角是一体两面：定位定义了你要满足哪种需求，需求三角验证这种需求是否真实。

→ **POEM营销模型（MK-2）**：定位是POEM中E（企业营销）的核心内容——企业营销的所有动作都应该围绕一个清晰的定位来展开。没有定位的营销是散弹枪，有定位的营销是狙击枪。

**7. 其他**

定位理论最常被误解的一点：定位不是「定位自己」，而是「在顾客心智中定位」。很多人的定位陈述是这样的：「我们是一家科技公司，专注于用AI赋能教育。」这不是定位，这是自我描述。定位是站在顾客角度提出的问题：在顾客的脑子里，你占据了哪个词？如果顾客想到那个词时想到的不是你，你的定位就还没有成立。

里斯说过一句话值得每个营销人贴在工位上：「定位不是你对产品做什么，定位是你对顾客的心智做什么。」产品的改进是长期的、渐进的，心智的改变是瞬间的、整体的一次转身。定位理论教你的不是如何做更好的产品，而是如何让顾客在需要这类产品时第一个想到你——这两件事有时重叠，但经常不重叠。

## 次要模型速览

### MK-2 POEM营销模型

**1. 一句话理解**

顾客的购买决策受四个因素影响——他自己的感受（P）、别人的评价（O）、企业的营销（E）、外部环境（E），四力同向时效果最强。

**2. 模型提出者**

整合营销传播框架的演进产物，将影响消费者决策的力量系统归类。POEM四个字母分别代表：Personal perception（个人感知）、Others' evaluation（他人评价）、Enterprise marketing（企业营销）、External environment（外部环境）。

**3. 模型介绍/概念**

消费者不是在真空中做决策的，他们同时被四股力量推拉。P是消费者自己的体验和印象——产品好不好用、品牌印象如何；O是他人的评价——朋友推荐、网红种草、差评劝退；E是企业的营销动作——广告、促销、品牌活动；E是外部环境——政策变化、文化趋势、宏观经济。其中外部环境是前提和边界：你可以改变P、O、E，但你改变不了E（外部环境）。最聪明的营销策略是让四力同向——比如新能源汽车，政策补贴（E）+口碑传播（O）+广告教育（E企业营销）+试驾体验（P），四股力量共同推动消费者决策。E（外部环境）与本书的反作用力模型（PC-6）直接呼应：不要在外部环境的反方向上用力，逆势营销代价极大、收效极微。

**4. 核心要素**

① 四力共同作用，单一力量效果有限；② 外部环境是不可控的「大势」，最好的策略是顺势；③ 当四力同向时，产生乘法效应——不是1+1+1+1=4，而是1×1×1×1=1（任何一个为零则效果归零）；④ E（外部环境）是其他三个力的前提，外部环境决定了P、O、E企业能发挥多大作用。

**5. 实践/运用领域**

营销策略制定（评估四力是否同向）、品牌口碑管理（O的力量越来越大）、市场进入分析（先看E，再布局P、O、E）。

**6. 与其他模型关系**

→ **反作用力（PC-6）**：POEM中的E（外部环境）就是大势——顺风事半功倍，逆风事倍功半，不要在反方向上用力。→ **SWOT分析（BU-2）**：POEM的E对应SWOT中的机会和威胁。→ **定位理论（MK-1）**：定位是E（企业营销）的核心——所有营销动作都围绕定位展开。

**7. 其他**

POEM提醒营销人一个常被忽略的事实：企业营销（E）只是四股力量之一，而且往往不是最强的那股。在社交媒体时代，O（他人评价）的影响力已经远超传统广告。与其花巨资投放没人信的广告，不如把预算转向口碑和体验——让用户自己成为营销力量。

### MK-3 需求三角

**1. 一句话理解**

需求=缺乏感+目标物+消费者能力——缺一样就不是真实需求。这是一把验证需求的筛子，任何创业想法都能用它过一遍。

**2. 模型提出者**

营销学中需求分析的系统化框架，整合了消费者行为学、行为经济学和心理学的核心洞见。

**3. 模型介绍/概念**

需求三角指出一个真实需求必须同时满足三个条件：缺乏感（痛点——用户感到缺了什么）、目标物（解决方案——什么能消除这个痛点）、消费者能力（用户愿意且能够支付的成本）。三者是乘法关系——任何一个为零，需求就不存在。有痛点没能力是空想（很多创新产品死于定价过高），有能力没痛点是伪需求（很多创业项目死于没人真的需要），有痛点有能力但没有目标物是空白市场（消费者知道痛但不知道有什么可以解决）。需求三角的乘法关系与第9章的Fogg行为设计学（SO-1，行为=动机×能力×触发）形成直接呼应：缺乏感对应动机，消费者能力对应能力，目标物对应触发——两个模型从不同学科独立推导出了同一个结构：行为的产生需要多个必要条件同时满足，缺一不可。

**4. 核心要素**

① 缺乏感——对现状的不满和缺憾，是需求的起点；② 目标物——具体的解决方案，是需求的出口；③ 消费者能力——支付意愿和支付能力，是需求的闸门；④ 三者是乘法关系——任何一个为零，整体为零。

**5. 实践/运用领域**

产品定义（验证需求是否真实）、创业（避免做伪需求产品）、营销策划（精准定位目标人群——他们有缺乏感吗？付得起吗？）。

**6. 与其他模型关系**

→ **Fogg行为设计学（SO-1）**：需求三角是Fogg模型在营销领域的镜像——缺乏感=动机，消费者能力=能力，目标物=触发，乘法结构完全对应。→ **设计思维（DE-1）**：设计思维中的同理心阶段就是发现缺乏感的过程。→ **定位理论（MK-1）**：定位定义目标物——你在需求三角中选哪个缺乏感来满足。

**7. 其他**

需求三角最大的价值不是帮你找到需求，而是帮你排除伪需求。每个创业者都觉得自己看到了需求——但用需求三角一过筛，大量「需求」其实只是创办者自己的想象。缺乏感够不够强？用户愿不愿意为此付费？有没有更便宜的替代方案？这三个问题能帮你省下九成的试错成本。

## 内在联系

本章三个模型构成一条从战略到验证到执行的完整链条：

**定位理论回答「在哪里竞争」。** 它告诉你应该在消费者心智中占据哪个位置——选哪个品类、绑哪个词、做谁心中的第一。这是战略层的选择：方向对了，后面的努力才有意义。

**需求三角回答「是否存在真实需求」。** 它验证你在那个心智位置上所瞄准的需求是否真实——缺乏感够不够强？目标物够不够明确？消费者有没有能力？这是验证层的工具：避免在一个看起来很美但实际不存在需求的方向上浪费资源。

**POEM回答「如何影响消费者决策」。** 它指导你如何调动四股力量推动消费者完成购买——个人感知、他人评价、企业营销、外部环境是否同向？这是执行层的框架：把定位和需求验证转化为具体的营销行动。

三者的逻辑链条是：定位选定心智位置（战略）→ 需求三角验证该位置上的需求真实性（验证）→ POEM设计影响决策的执行方案（执行）。跳过任何一步都会出问题——没有定位的营销是散弹枪；没有需求验证的定位是自嗨；没有POEM框架的执行是乱枪打鸟。

更深一层：定位理论选定的方向，决定了需求三角中「目标物」的定义——你选择占据哪个品类词，就意味着你选择满足哪种缺乏感；而需求三角的乘法结构又决定了POEM中四股力量必须同向——如果需求三角有任何一个要素为零，POEM的四力再同向也是零乘以一，结果还是零。三个模型环环相扣，缺一不可。

🔗 **跨学科连结**

- 需求三角连结Fogg行为设计学（SO-1）——两者都是「乘法关系、缺一不可」的结构，分别从营销和行为设计的角度说明了同一个道理：行为的产生需要多个必要条件同时满足。
- POEM中的E（外部环境）连结反作用力模型（PC-6）——不要在环境的反方向上用力，顺势营销事半功倍。
- 定位理论连结护城河（IN-3）——占据心智位置就是最强的品牌护城河，营销视角的构建方法和投资视角的估值方法互为补充。
- 需求三角的乘法结构连结概率论中的独立事件——每个要素为零是独立事件，整体为零的概率是各要素为零概率的并集，远大于单一要素为零的概率。

📋 **实战练习**

1. 用需求三角分析你正在做或想做的产品：缺乏感是什么？目标物是什么？消费者有能力支付吗？三者中最弱的是哪个？
2. 为你的产品/服务想一个定位——完成这句话：「说到___就想到我们」。如果填不出来，你的定位还不够锋利。
3. 画出你产品的POEM四力图：P、O、E（企业营销）、E（外部环境）分别是什么？四力是否同向？哪股力量最弱？
4. 找一个你熟悉的品牌，分析它的定位是否清晰——它占了哪个品类词？这个定位曾经变化过吗？变化带来了什么后果？

📊 **概率工具箱**

需求三角的乘法关系意味着：即使缺乏感、目标物、消费者能力各自成立的概率都很高（比如各0.9），需求成立的概率也只有0.9×0.9×0.9=0.729——不到四分之三。如果任何一个要素成立的概率只有0.5，整体概率就暴跌到0.125。这告诉我们：验证需求时，三个要素都要验证，而且都要高标准验证——仅验证一个就下结论，失败概率极高。

💰 **边际成本与机会成本分析**

- 用需求三角验证一个产品想法的边际成本是30分钟思考，边际收益是避免投入数万元甚至数年时间做一个伪需求——投资回报率可能是你做过的最高的一笔投资。
- 定位的边际成本是找到那个心智空位——这需要观察和思考而非烧钱。定位不清就大规模投放广告，机会成本是所有广告费都打了水漂。
- POEM分析的边际成本是画一张四力图，边际收益是发现哪股力量在拖后腿——对症下药比全面撒网高效得多。

📝 **可复用模板**

```
需求验证清单（需求三角）：
1. 缺乏感（痛点）：用户在烦恼什么？这个痛点有多强（1-10分）？
2. 目标物（解决方案）：我的方案如何消除痛点？用户能一眼看懂吗？
3. 消费者能力：用户愿意付多少钱？有没有更便宜的替代方案？
4. 三者中哪个最弱？能否补强？不能补强的，就该放弃。
```

```
定位一句话测试：
「说到 [品类词] 就想到 [品牌名]」
——品类词够锋利吗？（「饮料」不够，「0糖气泡水」够）
——品牌是第一个占据这个词的吗？
——如果不是第一个，能创造一个新品类吗？
```

```
POEM四力诊断：
P（个人感知）：用户用过后的感受如何？复购率多少？
O（他人评价）：口碑如何？NPS分数？社交讨论度？
E（企业营销）：广告、促销、品牌活动的核心信息是否一致？
E（外部环境）：政策、文化、经济是顺风还是逆风？
→ 四力是否同向？哪股力量最弱？如何补强？
```
