# 第10章 营销学：需求如何被发现和满足

---

📍 **破除焦虑**：「营销就是打广告和做活动吧？」→ 营销的核心不是推销，是理解需求——缺乏感+目标物+消费者能力=需求，三要素缺一不可。大多数人把营销等同于推广，但推广只是最后一个动作。真正的营销在产品诞生之前就开始了：找到谁有痛点，设计什么来消除痛点，确认对方付得起。做不到这三步，再多的广告预算也是打水漂。

💡 **一句话心法**：有痛点没能力是空想，有能力没痛点是伪需求——真实需求三者缺一不可。

📊 **学科之眼**：营销学教我们理解需求如何被发现和满足——从用户痛点到产品定位。它不教你「怎么卖」，它教你「先搞清楚有没有人想买」。
