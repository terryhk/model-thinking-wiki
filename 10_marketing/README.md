# 第10章 营销学：需求如何被发现和满足

---

📍 **破除焦虑**：「营销就是打广告和做活动吧？」→ 营销的核心不是推广，是理解需求——缺乏感+目标物+消费者能力=需求，三要素缺一不可。大多数人把营销等同于推广，但推广只是最后一个动作。真正的营销在产品诞生之前就开始了：找到谁有痛点，设计什么来消除痛点，确认对方付得起。做不到这三步，再多的广告预算也是打水漂。

💡 **一句话心法**：有痛点没能力是空想，有能力没痛点是伪需求——真实需求三者缺一不可。

📊 **学科之眼**：营销学教我们理解需求如何被发现和满足——从用户痛点到产品定位。它不教你「怎么卖」，它教你「先搞清楚有没有人想买」。

## 内在联系

本章三个模型构成一条从战略到验证到执行的完整链条：

**定位理论回答「在哪里竞争」。** 它告诉你应该在消费者心智中占据哪个位置——选哪个品类、绑哪个词、做谁心中的第一。这是战略层的选择：方向对了，后面的努力才有意义。

**需求三角回答「是否存在真实需求」。** 它验证你在那个心智位置上所瞄准的需求是否真实——缺乏感够不够强？目标物够不够明确？消费者有没有能力？这是验证层的工具：避免在一个看起来很美但实际不存在需求的方向上浪费资源。

**POEM回答「如何影响消费者决策」。** 它指导你如何调动四股力量推动消费者完成购买——个人感知、他人评价、企业营销、外部环境是否同向？这是执行层的框架：把定位和需求验证转化为具体的营销行动。

三者的逻辑链条是：定位选定心智位置（战略）→ 需求三角验证该位置上的需求真实性（验证）→ POEM设计影响决策的执行方案（执行）。跳过任何一步都会出问题——没有定位的营销是散弹枪；没有需求验证的定位是自嗨；没有POEM框架的执行是乱枪打鸟。

更深一层：定位理论选定的方向，决定了需求三角中「目标物」的定义——你选择占据哪个品类词，就意味着你选择满足哪种缺乏感；而需求三角的乘法结构又决定了POEM中四股力量必须同向——如果需求三角有任何一个要素为零，POEM的四力再同向也是零乘以一，结果还是零。三个模型环环相扣，缺一不可。

---

🔗 **跨学科连结**

- 需求三角连结Fogg行为设计学（SO-1）——两者都是「乘法关系、缺一不可」的结构，分别从营销和行为设计的角度说明了同一个道理：行为的产生需要多个必要条件同时满足。
- POEM中的E（外部环境）连结反作用力模型（PC-6）——不要在环境的反方向上用力，顺势营销事半功倍。
- 定位理论连结护城河（IN-3）——占据心智位置就是最强的品牌护城河，营销视角的构建方法和投资视角的估值方法互为补充。
- 需求三角的乘法结构连结概率论中的独立事件——每个要素为零是独立事件，整体为零的概率是各要素为零概率的并集，远大于单一要素为零的概率。

📋 **实战练习**

1. 用需求三角分析你正在做或想做的产品：缺乏感是什么？目标物是什么？消费者有能力支付吗？三者中最弱的是哪个？
2. 为你的产品/服务想一个定位——完成这句话：「说到___就想到我们」。如果填不出来，你的定位还不够锋利。
3. 画出你产品的POEM四力图：P、O、E（企业营销）、E（外部环境）分别是什么？四力是否同向？哪股力量最弱？
4. 找一个你熟悉的品牌，分析它的定位是否清晰——它占了哪个品类词？这个定位曾经变化过吗？变化带来了什么后果？

📊 **概率工具箱**

需求三角的乘法关系意味着：即使缺乏感、目标物、消费者能力各自成立的概率都很高（比如各0.9），需求成立的概率也只有0.9×0.9×0.9=0.729——不到四分之三。如果任何一个要素成立的概率只有0.5，整体概率就暴跌到0.125。这告诉我们：验证需求时，三个要素都要验证，而且都要高标准验证——仅验证一个就下结论，失败概率极高。

💰 **边际成本与机会成本分析**

- 用需求三角验证一个产品想法的边际成本是30分钟思考，边际收益是避免投入数万元甚至数年时间做一个伪需求——投资回报率可能是你做过的最高的一笔投资。
- 定位的边际成本是找到那个心智空位——这需要观察和思考而非烧钱。定位不清就大规模投放广告，机会成本是所有广告费都打了水漂。
- POEM分析的边际成本是画一张四力图，边际收益是发现哪股力量在拖后腿——对症下药比全面撒网高效得多。

📝 **可复用模板**

```
需求验证清单（需求三角）：
1. 缺乏感（痛点）：用户在烦恼什么？这个痛点有多强（1-10分）？
2. 目标物（解决方案）：我的方案如何消除痛点？用户能一眼看懂吗？
3. 消费者能力：用户愿意付多少钱？有没有更便宜的替代方案？
4. 三者中哪个最弱？能否补强？不能补强的，就该放弃。
```

```
定位一句话测试：
「说到 [品类词] 就想到 [品牌名]」
——品类词够锋利吗？（「饮料」不够，「0糖气泡水」够）
——品牌是第一个占据这个词的吗？
——如果不是第一个，能创造一个新品类吗？
```

```
POEM四力诊断：
P（个人感知）：用户用过后的感受如何？复购率多少？
O（他人评价）：口碑如何？NPS分数？社交讨论度？
E（企业营销）：广告、促销、品牌活动的核心信息是否一致？
E（外部环境）：政策、文化、经济是顺风还是逆风？
→ 四力是否同向？哪股力量最弱？如何补强？
```
