# 第12章 投资学：所有学科的综合考场

---

📍 **破除焦虑**：「投资学就是选股和看K线吧？我不炒股不需要学。」→ 投资学的本质不是炒股，是在不确定中做决策——能力圈、护城河、市场先生、格栅理论，这些模型适用于人生的每一个重大决策。

💡 **一句话心法**：知道自己能力圈的边界，比扩大能力圈更重要——你不一定需要更大的圈子，你需要的是清楚地知道圈子的边际在哪里。

📊 **学科之眼**：投资学是所有学科的综合考场——物理的法则、数学的概率、心理的偏差、经济的逻辑、商业的判断全部交汇，共同决定你的收益或损失。
